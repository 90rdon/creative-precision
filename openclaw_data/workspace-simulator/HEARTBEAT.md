# HEARTBEAT.md — Simulator Agent

## Periodic Tasks (Adversarial Training)

When triggered by the OpenClaw cron (e.g., nightly at 2:00 AM), run the following:

1. **Review Latest Weaknesses:** Read the most recent `docs/reports/GTM_REPORT_*.md` generated by the Synthesizer.
2. **Formulate Strategy:** Decide if tonight is an "Exploit" night (hammering a specific weak trait) or an "Explore" night (testing random edge-case personas).
3. **Run Simulation:**
   ```bash
   cd server && npx ts-node src/api/openclaw/simulator.ts run --mode=auto
   ```
4. **Log Run:** Record a short summary of the personas deployed in `memory/simulations.md`.
